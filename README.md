# segunda_aula-curso-Guanabara
Segunda aula do curso Guanabara em JS &lt; Java Script

```
<!DOCTYPE html>
<html>

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>segunda aula do curso do Guanabara</title>

</script>
</body>

</html>
```
